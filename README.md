# 🏥 SwasthNova

> **AI-Powered Multilingual Healthcare Operations & Resource Intelligence Platform**

SwasthNova is a next-generation AI-powered healthcare management platform that enables District Health Administrators, Primary Health Centres (PHCs), Community Health Centres (CHCs), and hospitals to monitor healthcare operations in real time. The platform leverages **Google Gemini AI** to predict medicine shortages, forecast demand, optimize resource allocation, and support faster, data-driven healthcare decisions.

## 🌐 Live Demo

**https://swasth-nova.netlify.app/**

---

# 📖 Overview

SwasthNova centralizes healthcare operations into a single intelligent dashboard, providing real-time visibility into medicine inventory, vaccine availability, patient footfall, bed occupancy, doctor attendance, laboratory services, and overall health centre performance.

Using AI-powered predictive analytics, the platform helps healthcare administrators proactively manage resources, reduce medicine shortages, improve operational efficiency, and deliver better patient care.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/SwasthNova.git
```

Navigate to the project

```bash
cd SwasthNova
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

# 🔐 Environment Variables

Create a `.env` file.
